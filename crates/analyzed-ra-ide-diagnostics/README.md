# analyzed-ra-ide-diagnostics

The analyzed build of [`ra_ap_ide_diagnostics`](https://crates.io/crates/ra_ap_ide_diagnostics). The build script unpacks the checksum-verified upstream source from crates.io, patches it at build time, and compiles the result; the package ships no upstream code.

Part of [analyzed](https://github.com/zetaloop/analyzed), a shared rust-analyzer daemon.
